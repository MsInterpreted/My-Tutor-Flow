HOW TO ADD IMAGES TO THE PLATFORM
====================================================

Step 1: Generate each image using the prompt above.
        Recommended tools: Midjourney, DALL-E 3 (ChatGPT Plus), Adobe Firefly
        Size: 800×500px landscape (16:9 or wider crop)
        Format: .jpg or .webp

Step 2: Save the image to:
        C:/Users/dalz2/edubooks-platform/public/images/[filename].jpg

Step 3: Open the matching data file and add an  image  field to that unit:

        Example — beginner Unit 1:
        Open: src/data/comprehension/beginner.js
        Find: id: 1,
        Add below it:  image: '/images/comprehension-beginner-unit-1.jpg',

Step 4: Rebuild and redeploy:
        npm run build
        Drag dist/ to netlify.com/drop

The placeholder disappears automatically when the image field is present.
If the image fails to load (wrong filename etc.), the error state shows the filename to help debug.
