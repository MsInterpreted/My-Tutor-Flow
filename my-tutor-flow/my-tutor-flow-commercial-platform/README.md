## Overview

TD Learning Academy (TDLA) is a comprehensive tutoring management platform for running a professional tutoring business. Built with React 19, Vite 6, Material UI, and Firebase.

### Key Features

- **Student Management** - Complete student profiles with academic tracking
- **Attendance Tracking** - Session logging with duration and progress monitoring
- **Smart Billing System** - Multi-currency support (ZAR, GBP, USD, EUR, AED)
- **Advanced Analytics** - Business intelligence with revenue and performance insights
- **Mobile Optimized** - Fully responsive design for all devices
- **South African Education System** - Grades, subjects, terms aligned to SA curriculum
- **PDF Reports** - Generate and download professional reports
- **Dark Mode** - Full dark/light theme support

## Tech Stack

- **Frontend**: React 19, Vite 6, Material UI 7
- **Backend**: Firebase (Auth, Firestore, Storage)
- **Charts**: Recharts
- **PDF**: jsPDF, html2canvas

## Getting Started

```bash
# Install dependencies
npm install

# Copy environment template and configure
cp .env.template .env

# Start development server
npm run dev

# Build for production
npm run build
```

## First Time Setup

1. Enable Email/Password auth in Firebase Console
2. Navigate to `/signup`
3. Use authorization code `MASTER-TUTOR-2024` for admin access
4. Create your admin account
